# David Jones EOFY Marketing Analytics — SQL Database Design & BI

A relational database and analytics project modelling the full customer
journey for a David Jones end-of-financial-year marketing campaign.
Built as part of a Master of Business Analytics at Macquarie University.

## Overview
Designed a 12-entity MySQL database and wrote 10 SQL queries to deliver
CMO-level campaign insights from customer, transaction, and channel data.

## Key Insights
- Facebook delivered 93.75% click-to-conversion vs 20% on X (Twitter)
- Platinum loyalty members generated 34x more revenue than Bronze members
